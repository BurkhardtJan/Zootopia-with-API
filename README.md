# Zootopia with API

This is an MSIT Coding School Project.
It fetches animal data from API Ninjas and returns a website.

## Installation

To install this project, simply clone the repository and install the dependencies in requirements.txt using `pip`.

Create an API Key on https://api-ninjas.com/.
Create a file named .env and insert 
```
API_KEY = "YourAPIKey"
```

## Usage

To use this project, run the following command - `python main.py`.
